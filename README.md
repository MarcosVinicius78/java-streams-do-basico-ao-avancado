# Java Streams – Do Básico ao Avançado

Este repositório contém meus estudos sobre a Stream API do Java,
indo do básico ao avançado, com exemplos práticos, exercícios
e pequenos projetos simulando cenários reais de backend.

## Conteúdo
- Fundamentos da Stream API
- Operações intermediárias e terminais
- Streams com objetos
- Collectors e agrupamentos
- Reduce, Optional e FlatMap
- Comparação entre Stream, for e for-each
- Projetos práticos

## Tecnologias
- Java 8+
- IntelliJ IDEA / VS Code
- Git e GitHub

## Objetivo
Aprender Stream API na prática, escrevendo código limpo,
legível e próximo do uso real em projetos Java/Spring.
